## 运行时数据区域

### 程序计数器

  线程私有
  程序计数器是一块较小的内存空间。它可以看作是当前线程所执行的字节码的行号指示器。

### Java虚拟机栈

  线程私有。
  生命周期与线程相同。
  每个方法创建一个栈桢。(局部变量表，操作数栈，动态链接，方法出口等信息)
  局部变量表：存放了编译期可知的各种基本数据类型(boolean, byte, char, short, int, float, long, double),
  对象引用(reference 类型）

### 本地方法栈

  线程私有。
